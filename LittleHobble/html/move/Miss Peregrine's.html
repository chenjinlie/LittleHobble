月光透过云彩，刚好照亮我脚下的路

片名/佩小姐的奇幻城堡

导演: 蒂姆·波顿

编剧: 简·古德曼 / 兰森·里格斯

主演: 阿萨·巴特菲尔德 / 伊娃·格林 / 塞缪尔·杰克逊 / 朱迪·丹奇 / 艾拉·珀内尔 / 克里斯·奥多德 / 金·迪肯斯 / 艾莉森·珍妮 / 劳伦·麦克罗斯蒂 / 特伦斯·斯坦普 / 鲁伯特·艾弗雷特 / 芬利·麦克米伦 / 艾拉·瓦勒斯泰特 / 米洛·帕克 / 匹克西·戴夫斯 / 澳澜·琼斯

剧情简介：

杰克（阿萨·巴特菲尔德 Asa Butterfield 饰）的爷爷自杰克还小的时候，就会常说床边故事给他听，故事中有许多拥有特殊能力的小孩。爷爷神秘死亡后，留下了关于这个属于另一个时空世界种种谜团的线索，杰克循着这些线索，进入了这个神秘的世界，发现这群奇怪的孩子真实存在，他们住在这里是为了躲避可怕的变异怪兽。随着杰克的到来，危险也同时抵达，杰克需要与这群孩子一起，抵御黑暗力量。

这样的一个童话故事却有着背后有点暗黑的寓意，也是让人感觉不太舒服了。相信看过本片的人都会有这样的一个错觉，本片应该是《大鱼》的番外篇。而且这个番外有点暗。

同样是自己的长辈的故事，无疑是将自己的父亲变为了自己的爷爷，因为这样一来故事从2016年开始计算往前推一推应该是可以到达二战时期了。因此故事里面的人无设定才会有意义。其实我们并不会第一次观影的时候对于之前的铺垫有任何的兴趣的。因为我们知道我们只不过是在等待一个故事的开始。

随着最终的铺垫落下帷幕，主人公终于成功的走进了那个故事里面。这并不是一个只有爱的故事，很多时候我们发现这个故事里面还有其他的。超越爱而存在的东西，比如说亲情，比如说责任。人们有序的生活着然后开心的起舞，仿佛这是一个世外桃源般的存在。这里我们一定会认为这不就是《大鱼》么？

但是真的不是的。如果这是《大鱼》也就好了。也就不会有之后的暗黑了。当我们看完通篇之后何尝不想这个故事是一个《大鱼》呢？但是这已经成为了一种奢望。故事总归不会一直像描述的那样阳光灿烂的。因为有黑暗将要侵袭这个世外桃源。而这个怪物就在这个故事本应该向着美好方向发展的时候出现了。很多时候总是希望自己永远不要知道美好时光背后的事情，但是我们却时时刻刻被好奇心驱使着，一步一步走向真相。真相虽然并不残酷但是很多时候都在不停的拷问着我们。因为知道真相的我们就会变得小心翼翼。就会变得心事重重。

即使是这样我们页依旧是停不下来自己的好奇心的。直到真相被解读，我们才会意识到自己如果当时没有知道那该多好。但是故事就是故事，一旦真相被揭开，那么厄运也就随之到达。怪物出现了。怪物有着自己的目标，而阳光下的美好的人们将面临严峻的考验。本该风风平浪静的世外桃源已然发生了变化。

我对于男主的出现是不报有任何的好感的，因为一般情况下一个宁静的环境被打破之后我们往往看到的是另一种幻灭。一定程度上这个男主的出现推动了情节的发展，但是另一方面片中的美好的时光就一去不复返了。因为本片并不是像《大鱼》那样的片子。因此可以得知一切的灾难基本上是由男主的好奇心推动的。这就尴尬了。

至于为什么怪物要吃小孩的眼睛，我是这样认为的，因为一旦小孩看不见。他们就会从童话中消失，永远回不去了。因为不能看见很多东西，因此想象力也就失去了，也就逐渐的成为了大人了。而且在143年的大环境下。本片最后的光明结局更加预示着二战的结束的号角已经吹响。即使是纳粹泯灭了人们的梦想但是人们依旧是可以看到希望的。胜利总会到来只不过期间的过程略微艰辛了一些。

抛开一些寓意来说本片是一部极富想象力的奇幻电影，仅仅是从片中那些身怀绝技的人物的设定就可见一斑。而且同时兼顾了人们对于美好时光的留恋。因此可以形成一个美好的日子的永久性循环。这不失为一种期望和向往。

有时候梦很美，以至于让你不愿意醒来。虽然现实会呼唤你回去，但是你依旧不愿意就这样离开。因为只有在梦里，我爱的人才不会老去。
